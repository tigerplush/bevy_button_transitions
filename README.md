# bevy_button_transitions

A simple button transition component (less than 100 lines of code). Supports image swapping and color tinting like Unity does.

## Supported Bevy Versions

| Bevy | bevy_button_transitions |
| ---- | ----------------------- |
| 0.15 | 0.1                     |
